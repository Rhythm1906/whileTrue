# Judge's Court - AI Courtroom Decision Game

An AI-powered courtroom game where you play as a judge making decisions on dynamic cases.

## Tech Stack
- **Frontend:** React + Vite + Firebase
- **Backend (Optional ML):** Python Flask + Custom Model
- **AI:** Google Gemini API
- **Auth/DB:** Firebase
- **Hosting:** GitHub Pages

## Features
- Google Authentication
- Save/Load game progress
- AI-generated cases via Gemini
- Local fallback cases (offline support)
- Three stat system: Trust, Economy, Chaos
- Optional ML model for case difficulty prediction
